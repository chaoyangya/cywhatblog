1.新建Django项目

安装Django和Mysql

1
2
3

pip3 install Django==1.11.7   #其他版本也可,Django后不加版本即可安装最新版

pip3 install pymysql

构建web应用

python3  manage.py startapp web

DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.mysql',  # 默认
        'NAME': 'XXXX',  # 连接的数据库名
        'HOST': 'localhost',  # mysql的ip地址
        'PORT': 3306, # mysql的端口
        'USER': 'root',  # mysql的用户名
        'PASSWORD': 'XXXX'  # mysql的密码
    }
}
